Votre installeur est prêt !
Le fichier install-lmdv.ps1 a été téléchargé.
  1. Copiez le fichier sur le PC Windows de votre caisse (clé USB ou email)
  2. Sur le PC Windows : clic droit sur le fichier → Exécuter avec PowerShell
  3. Si un avertissement de sécurité s'affiche, cliquez Exécuter quand même
  4. L'installation est entièrement automatique — comptez 2 à 5 minutes
  5. À la fin, l'interface admin s'ouvre automatiquement dans le navigateur

La clé API n'est jamais envoyée à un serveur — elle est inscrite directement dans le script local que vous téléchargez.
Ce script s'exécute uniquement sur le PC de votre caisse.
